Install Cart-It

Cart-It is listed on Microsoft Edge Add-ons and Mozilla (AMO). Users install in one click from those pages when you paste your public listing URLs into STORE_URLS below (Edge store URL + Firefox add-on URL). Until then, developers can use the zip with Load unpacked.

Download extension (zip)

Zip path (advanced): Edge → edge://extensions · Firefox → about:debugging#/runtime/this-firefox → Load Temporary Add-on (or use Load unpacked in Firefox Developer Edition). Turn on Developer mode where shown, then load the folder that contains manifest.json.

Open your browser’s extensions page

Sites cannot silently install extensions (browser security). Use the store buttons above for true one-click installs. These shortcuts only open the extensions UI — useful after downloading the zip.

Steps (zip / unpacked)

  1. Unzip cart-it-extension.zip somewhere easy to find (Desktop is fine).
  2. Open the extensions page using the buttons above (same browser you’re loading into).
  3. Turn on Developer mode where applicable.
  4. Load unpacked / Load Temporary Add-on and pick the folder that contains manifest.json.
  5. Log in on this site — the extension syncs your session from an open tab.

Tip: Set STORE_URLS.edge and STORE_URLS.firefox so visitors see the two green store buttons — that’s the real one-click install for Edge and Firefox.
