🛒 cart-It

Download extension ✨

Compatible with Microsoft Edge and Google Chrome. Who this page is for: you (and anyone helping you build Cart-It) — not your final shoppers yet. After you publish the extension to the Edge / Chrome store, customers will install with one click and won’t need Developer mode.

Why Developer mode? Browsers only let you “Load unpacked” from a folder on your computer when that safety switch is on. It’s their rule for anything that isn’t coming from the official store.

Right now there’s no store package — so pick your browser below and follow the steps to wire up the same extension folder you have in your project.

What to do (same idea for both browsers)
  1. Click the Download / add button for Edge or Chrome — it copies the right setup link.
  2. Paste into your browser’s address bar, press Enter.
  3. Turn on Developer mode → Load unpacked → choose your project’s extension folder.

Compatible with: Microsoft Edge

Use this block to download / add cart-It in Edge.

  1. Click the orange button (copies edge://extensions).
  2. Paste in the address bar → Enter.
  3. Developer mode on → Load unpacked → select the folder with manifest.json.
← Step 1 for Edge: copies the extensions page link

Compatible with: Google Chrome

Use this block to download / add cart-It in Chrome.

  1. Click the button (copies chrome://extensions).
  2. Paste in the address bar → Enter.
  3. Developer mode → Load unpacked → same extension folder.
← Step 1 for Chrome: copies the extensions page link

Which folder to choose?

When it asks for a folder, choose:

…\Cart_It\extension\  ← this folder (must include manifest.json)

After it’s loaded, open cart-It in the browser and log in — the extension will sync your session. Close this tab when you’re done.
