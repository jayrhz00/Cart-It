Install Cart-It

Published extensions install in one click from the store. Until then, use the zip below — two short steps after download.

Add to Chrome Edge Add-ons Firefox Add-ons
Download extension (zip)

Unzip the folder, then in Chrome or Edge: chrome://extensions or edge://extensions → Developer mode on → Load unpacked → pick the unzipped folder.

Open extensions page

Browsers block websites from installing extensions automatically (security). These buttons jump to the right page; you still choose Load unpacked after downloading the zip above.

Steps (after download)

  1. Unzip cart-it-extension.zip somewhere easy to find (Desktop is fine).
  2. Open the extensions page using the buttons above.
  3. Turn on Developer mode (toggle in the corner).
  4. Click Load unpacked and select the unzipped folder (the one that contains manifest.json).
  5. Log in on this site — the extension syncs your session from an open tab.

Tip: Paste your Chrome, Edge, and Firefox listing URLs in STORE_URLS below so visitors get one-click installs from each store.
